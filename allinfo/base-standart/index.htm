Способы взаимодействия Фонда с получателями финансовых услуг

Получатель финансовых услуг вправе обратиться в Фонд с использованием удобного для него способа связи:

  1. лично по адресу: г. Сургут, ул. Энтузиастов 52/1;
  2. по телефону;
  3. по электронной почте;
  4. с помощью почтовой связи;
  5. через личный кабинет.

Кроме того, получатель финансовых услуг вправе согласовать способ связи Фонда с ним. Для этого необходимо заполнить заявление, распечатать, подписать его и предоставить в Фонд удобным способом:

  • лично или направить по почте;
  • отсканировать подписанное заявление и направить его на адрес электронный почты Фонда.
